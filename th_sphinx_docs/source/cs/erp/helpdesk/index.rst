HELPDESK
========

Tổng quan
---------

A. Mô tả module
~~~~~~~~~~~~~~~
Module Helpdesk là hệ thống quản lý và xử lý các yêu cầu hỗ trợ từ người dùng. Module này giúp tổ chức theo dõi, phân công và giải quyết các vấn đề kỹ thuật,
yêu cầu hỗ trợ một cách hiệu quả.

B. Đối tượng sử dụng
~~~~~~~~~~~~~~~~~~~~
Nhân viên công ty có nhu cầu được hỗ trợ kỹ thuật

C. Phụ thuộc (các module liên quan)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
1. base
2. mail
3. helpdesk
4. documents_spreadsheet
5. th_maintenance
6. th_setup_parameters

D. Chức năng chính
~~~~~~~~~~~~~~~~~~
1. Quản lý phiếu hỗ trợ: Tạo, theo dõi và xử lý các yêu cầu hỗ trợ
2. Báo cáo: Phân tích phiếu và phân tích đánh giá
3. Cấu hình: Thiết lập khu vực hỗ trợ, danh mục, đặc điểm danh mục, chủ đề, nhóm hỗ trợ, giai đoạn, SLA

E. Các quyền trong module
~~~~~~~~~~~~~~~~~~~~~~~~~
1. Người dùng
2. Nhân viên hỗ trợ
3. Quản lý hỗ trợ
4. Quản trị viên
