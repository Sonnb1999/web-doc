APM
===

Tổng quan
---------

A. Mô tả module
~~~~~~~~~~~~~~~
Module APM (AUM Product Management) là hệ thống quản lý sản phẩm ngắn hạn dành cho AUM. Mục tiêu chính là hỗ trợ:

1. Quản lý cơ hội bán hàng
2. Chăm sóc khách hàng
3. Quản lý đơn hàng
4. Đồng bộ dữ liệu giữa các hệ thống liên quan

B. Đối tượng sử dụng
~~~~~~~~~~~~~~~~~~~~
1. Nhân viên phát triển mạng lưới
2. Nhân viên Marketing
3. Nhân viên trung tâm tuyển sinh
4. Nhân viên trung tâm vận hành
