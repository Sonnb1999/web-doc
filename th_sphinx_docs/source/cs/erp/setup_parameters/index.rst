SETUP PARAMETERS
================

Tổng quan
---------

A. Mô tả module
~~~~~~~~~~~~~~~

Module th_setup_parameters cho phép quản trị viên cấu hình các thông số hệ thống và thông tin nền tảng (đơn vị sở hữu, xuất xứ, ngành học, vùng - trạm, nhóm nguồn,  trạng thái, môn học, phân hệ, nhóm sản phẩm...). Giao diện trực quan, dễ thao tác, giúp chuẩn hóa dữ liệu đầu vào phục vụ các phân hệ khác như CRM, Helpdesk, Tuyển sinh, v.v.

B. Đối tượng sử dụng
~~~~~~~~~~~~~~~~~~~~

Quản trị viên (Administrator)
