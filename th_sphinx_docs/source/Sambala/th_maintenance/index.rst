MAINTENANCE
===========

Tổng quan
---------

A. Mô tả module
~~~~~~~~~~~~~~~
Module th_maintenance là module cho việc quản lý tài sản

1. Quản lý tài sản
2. Bảo trì
3. Sửa chữa
4. Kiểm kê

B. Đối tượng sử dụng
~~~~~~~~~~~~~~~~~~~~
1. Quản lý hành chính
2. Hạ tầng

C. Phụ thuộc (các module liên quan)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
1. base
2. maintenance
3. web_domain_field
4. th_employee

D. Chức năng chính
~~~~~~~~~~~~~~~~~~
1. Quản tài sản cá nhân, dùng chung
2. Quản lý cấp phát tài sản
3. Quản lý tài sản bảo trì
4. Quản lý tài sản sửa chữa
5. Quản lý kiểm kê
6. Cấu hình

E. Các quyền trong module
~~~~~~~~~~~~~~~~~~~~~~~~~

2. Trưởng nhóm
3. Quản trị viên
