FASTAPI
=======

Tổng quan
---------

A. Mô tả module
~~~~~~~~~~~~~~~
Th FastAPI là module dùng để cấu hình và customise các Endpoint nhằm mục đích triển khai Fast API cho hệ thống Sambala.

B. Đối tượng sử dụng
~~~~~~~~~~~~~~~~~~~~
1. Phòng phát triển ERP (Trung tâm IT)

C. Phụ thuộc (các module liên quan)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
1. fastapi
2. th_srm
3. th_setup_parameters

D. Chức năng chính
~~~~~~~~~~~~~~~~~~
1. Tạo cơ hội APM bên B2B
2. Xóa cơ hội APM bên B2B
3. Tạo đặc điểm APM sang B2B
4. Tạo cơ hội CRM bên B2B
5. Xóa cơ hội CRM bên B2B
6. Đồng bộ cơ hội CRM bên B2B
7. Tạo log note bên B2B
8. Xóa log note bên B2B
9. Tạo lịch sử chăm sóc bên B2B
10. Tạo thông báo cơ hội trùng bên B2B
11. Đồng bộ bảng Map ID bên B2B
12. Đồng bộ liên hệ bên B2B
13. Lấy thông tin sinh viên SRM bên B2C
